01 Frida 环境准备

Nov 26, 2024
阅读时长: 1 分钟

PC 端

  • 推荐使用 Linux 系统环境
  • Android Studio & SDK
  • Android Soursce Code
    1. androidxref 这个目前最高版本是 9.0
    2. aospxref 好像挂了
    3. aosp.app 目前最高版本是 14.0
  • 运维工具 htop, jnettop (Linux)

ROOT 手机

  • 解 Bootloader 锁
  • TWRP 刷入 Recovery 分区
  • 安装 Magisk
  • 刷 kali-nethunter 系统（有条件的可以试试，提供一个完整的 Linux 系统环境, 能使用strace等命令）

Frida

Frida 是平台原生App的 Greasemonkey，就是一种动态插桩工具，通过插入代码到原生App的内存空间去动态地监视和修改其行为。Greasemonkey 是 Firefox 的一套插件体系，通过利用 Greasemonkey 插入自定义的 JavaScript 脚本可以定制网页的显示或行为方式。 通俗来讲就是可以直接改变 Firefox 对网页的编排方式，从而实现想要的任何功能。

PC端直接使用 pip 安装

1
2
3

python -m venv venv
source venv/bin/activate
pip install frida-tools  # 自动安装最新版本的全系列工具

手机端安装对应版本的 frida-server。

  • 选择与 PC 端 frida 同版本；
  • 选择与 手机同架构的版本（arm, amd, x86, x64等）；

开发环境

  • IDE: VSCode吧（安装 python, js 相关插件，主要是能自动提示和补全 API）
  • 安装 nodejs 环境：不建议 apt | yum 等命令在线安装，版本可能会比较低，去 github 获取新版；
  • 从 Github 上下载模板工程 frida-agent-example
