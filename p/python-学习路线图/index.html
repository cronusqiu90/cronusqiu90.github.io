Python 学习路线图

May 31, 2015
阅读时长: 1 分钟

环境的搭建

基础知识

  • 语法、变量和类型、数字和字符串、运算符
  • 分支、循环结构
  • 函数、模块
  • 常用数据结构、字符串和正则表达式
  • 面向对象
    • 类和对象，属性和方法，抽象,封装，继承，多态等
    • 魔术属性和方法, mixin, metaclass
  • 文件和异常
  • 进程和线程
  • 网络操作
  • 高阶函数、Lambda、闭包、装饰器
  • 迭代起、生成器
  • 并发、异步

管理工具

  • 版本控制: svn, git
  • 运行环境: venv, virtualenv, pyenv
  • 依赖库: Pip, conda, Poetry
  • 打包发布： Wheel, Pypi
  • 代码格式化检查: isort, black, mypy, pylint
  • 注释文档： Sphinx
  • 测试工具: unitest, pyunit, pytest
  • 自动化工具: tox

中间件

  • 数据库：SQLAlchemy （sqlite, mysql）
  • 缓存: Redis, Memcached
  • 队列: pika (RabbitMQ)

框架

  • Web框架: Tornado, Django, Flask, FastAPI
  • 爬虫: Scrapy
  • 分布式框架：Celery

项目

  • 部署工具: ansible, fabric, docker
  • 自动化测试
  • 性能测试、调优

数据处理分析

  • NumPy, Pandas
  • 可视化: matplotlib

机器学习和深度学习

  • 基础知识
  • K最近邻分类，决策树，贝叶斯分类，向量机，回归分析 ……
  • PyTorch
