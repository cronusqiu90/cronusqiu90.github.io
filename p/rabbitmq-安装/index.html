RabbitMQ 安装

Mar 05, 2020
阅读时长: 2 分钟

Install RabbitMQ on CentOS 7

Update

1
2

$> sudo yum -y install epel-release
$> sudo yum -y update

Install Erlang

  1. Download repository
1

$> wget http://packages.erlang-solutions.com/erlang-solutions-1.0-1.noarch.rpm
  1. Add repository
1

$> sudo rpm -Uvh erlang-solutions-1.0-1.noarch.rpm
  1. Install erlang and dependencies
1

$> sudo yum -y install erlang socat logrotate

Install RabbitMQ

  1. Download RabbitMQ package
1

$> wget https://github.com/rabbitmq/rabbitmq-server/releases/download/v3.8.35/rabbitmq-server-3.8.35-1.el8.noarch.rpm
  1. Add signing key
1

$> sudo rpm --import https://www.rabbitmq.com/rabbitmq-signing-key-public.asc
  1. Install rabbitmq-server
1

$> sudo rpm -Uvh rabbitmq-server-3.8.35-1.el8.noarch.rpm
  1. Start RabbitMQ
1

$> sudo systemctl start rabbitmq-server
  1. Automatically start RabbitMQ at boot time
1

$> sudo systemctl enable rabbitmq-server

Firewall

  1. Open port
1
2

$> sudo firewall-cmd --zone=public --permanent --add-port=5672/tcp
$> sudo firewall-cmd --zone=public --permanent --add-port=15672/tcp
  1. Reload firewall
1

$> sudo firewall-cmd --reload

Enable RabbitMQ Web Management Plugin

  1. enable plugin
1

$> sudo rabbitmq-plugins enable rabbitmq_management
  1. Modify file permissions
1

$> sudo chown -R rabbitmq:rabbitmq /var/lib/rabbitmq/
  1. Create an admin user (Change password to a strong password)
1

$> sudo rabbitmqctl add_user admin [PASSWORD]
  1. Make admin user and administrator
1

$> sudo rabbitmqctl set_user_tags admin administrator
  1. Set admin user permissions
1

$> sudo rabbitmqctl set_permissions -p / admin ".*" ".*" ".*"
  1. Access the RabbitMQ admin
1

http://[HOST]:15672

Recommended: delete guest user
